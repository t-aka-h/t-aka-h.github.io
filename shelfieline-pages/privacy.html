Shelfieline — プライバシーポリシー

最終更新: 2026-05-09 (v1.0)

Shelfieline (以下「本アプリ」) は個人開発者が提供するアプリです。本ポリシーは、本アプリがユーザー情報をどのように扱うかを説明します。

1. データ収集

本アプリは、ユーザーの個人データを開発者のサーバーに送信・収集することはありません。蔵書情報・読書ステータス・読書日記・表紙画像など、すべてのデータは iOS 標準の SwiftData によって お使いの端末内にのみ 保存されます。

2. アカウント登録

本アプリはアカウント登録機能を持ちません。メールアドレス・電話番号・氏名などの登録は不要です。

3. ユーザートラッキング

本アプリは Google Analytics や Facebook SDK 等の第三者アナリティクス・トラッキング SDK を組み込んでいません。広告も表示しません。App Tracking Transparency (ATT) のリクエストも行いません。

4. 外部 API への通信

書誌情報 (タイトル・著者・出版社など) を取得するため、ユーザーがバーコードスキャン等で本を追加した際に、以下の公式 API へ ISBN を送信します:

  • openBD (https://openbd.jp/) — 書誌情報の自動取得
  • 国立国会図書館サーチ (https://ndlsearch.ndl.go.jp/) — 書誌情報の補完

これらのリクエストには ISBN 番号のみが含まれ、ユーザー識別子・端末識別子などの個人情報は一切送信されません。

5. カメラ・写真ライブラリの使用

本アプリは以下の用途で iOS のカメラ・写真ライブラリへのアクセスを求めます:

  • カメラ: 本のバーコード読み取り (Vision フレームワーク)、本の表紙をユーザー自身で撮影する用途
  • 写真ライブラリ: ユーザーが既に撮影/保存している表紙画像を本に紐付ける用途

取得された画像はすべて端末内 (SwiftData の external storage) にのみ保存され、外部へ送信されることはありません。

6. Share Extension

本アプリには Share Extension (BookshelflineShare) が含まれます。Safari など他アプリから「画像」または「URL」を共有した際、共有データは App Group 経由でメインアプリの inbox に一時保存され、ユーザーが本に紐付けるまで保持されます。inbox の内容は端末内のみで、外部へ送信されません。

7. データの削除

本アプリで作成したすべてのデータは、本アプリを iPhone / iPad からアンインストールすることで完全に削除されます。開発者がデータを保持することはありません。

8. 子どもの利用

本アプリは全年齢 (4+) 対象です。13 歳未満の子どもからの個人データを意図的に収集することはありません。

9. ポリシーの変更

本ポリシーは予告なく改訂される場合があります。重要な変更がある場合は、本ページに改訂日を記載してお知らせします。

10. お問い合わせ

本ポリシーに関するご質問は trim.hoist.6b@icloud.com までお寄せください。

Shelfieline — Privacy Policy

Last updated: 2026-05-09 (v1.0)

Shelfieline ("the App") is an iOS / iPadOS application provided by an independent developer. This Privacy Policy explains how the App handles user information.

1. Data Collection

The App does not transmit or collect any personal data to the Developer's servers. All data including library entries, reading status, reading notes, and cover images is stored solely on your device using Apple's SwiftData framework.

2. Account Registration

The App has no account registration feature. No email address, phone number, or personal identifiers are required.

3. User Tracking

The App contains no third-party analytics or tracking SDKs (no Google Analytics, no Facebook SDK, etc.). No advertisements are shown. The App does not present an App Tracking Transparency (ATT) prompt.

4. External API Communication

When you add a book via barcode scan or ISBN entry, the App queries the following official APIs by sending only the ISBN number:

  • openBD (https://openbd.jp/) — bibliographic information lookup
  • National Diet Library Search (https://ndlsearch.ndl.go.jp/) — bibliographic supplementation

These requests contain only the ISBN. No user identifiers, device identifiers, or any other personal information are sent.

5. Camera and Photo Library Use

The App requests access to the iOS camera and photo library for the following purposes:

  • Camera: reading book barcodes (via the Vision framework), and letting the user capture a book cover photo themselves.
  • Photo Library: letting the user attach an existing photo from their library to a book as a cover image.

All captured images are stored only on your device (in SwiftData's external storage) and are never transmitted externally.

6. Share Extension

The App includes a Share Extension (BookshelflineShare). When you share an image or URL from another app such as Safari, the shared data is temporarily stored in the main app's inbox via App Group and held until you assign it to a book. The inbox content stays on the device and is not transmitted externally.

7. Data Deletion

All data created by the App is completely removed when you uninstall the App from your iPhone or iPad. The Developer retains no data.

8. Children's Use

The App is rated for all ages (4+). It does not knowingly collect personal data from children under 13.

9. Changes to This Policy

This Policy may be updated without prior notice. Material changes will be indicated by updating the date at the top of this page.

10. Contact

If you have any questions about this Privacy Policy, please contact us at trim.hoist.6b@icloud.com.
